Что выведет программа? Объяснить вывод программы.

```go
package main

func main() {
	ch := make(chan int)
	go func() {
		for i := 0; i < 10; i++ {
			ch <- i
		}
	}()

	for n := range ch {
		println(n)
	}
}
```

Ответ:
```
0
1
2
3
...
8
9
deadlock

Цикл for n := range ch получает данные из канала до того момента, пока канал не будет закрыт.
Поэтому, как только в горутине завершается цикл происходит дедлок.
```
